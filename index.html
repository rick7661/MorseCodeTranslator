Welcome to Morse Code Signal Transmitter Project Page

Morse Code Signal Transmitter is a free iPhone application available on App Store

Features:

  • Translate ASCII text to Morse Code and transmit with flash light and sound
  • Tap your Morse Code with a virtual telegraph key

Open Source

Some of the source code are available in GitHub
